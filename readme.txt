Offer flexible shipping to your customers based on many rules.

== Description ==

WooCommerce RuleHook Connector Pro is a WooCommerce extension which allows you to set shipping prices based on many rules such as weight, price, dimensions, user roles and much more.


== Installation ==

1. Upload the `rulehook-connector` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WooCommerce -> Settings -> Shipping.
4. Add `RuleHook Connector` to the shipping zones.
